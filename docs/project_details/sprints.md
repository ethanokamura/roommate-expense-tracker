# Sprints
During this project, we have 3 sprints (single week). The goal is to split up and delegate tasks to reach the end goal of completing the app.

## First Sprint
During the first sprint the following tasks must be completed:
- Set up Flutter
- Set up Supabase
- Create schema
- Head of house
  - Create household
  - Read household
  - Update household
  - Delete household

## Second Sprint
During the second sprint the following tasks must be completed:
- Head of house
  - Create roommate
  - Read roommate
  - Update roommate
  - Delete roommate
- Expenses
  - Create expenses
  - Read expenses
  - Update expenses
  - Delete expenses
  - Delegate expenses by roommate (split costs)
  - Mark as paid
  - Mark as partially paid
- Due dates
